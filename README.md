## What is this?
Working paper proposing SAR: an AI safety monitoring framework addressing two documented vulnerabilities in stateful monitoring systems:
1. Collusion risk — same-lab monitors share blind spots
2. Iterative attack vulnerability — stateful judges accumulate exploitable memory

This is a testable hypothesis, not a solved problem.
